# ZM Collection Showcase

This is a public showcase repository for **ZM Collection**. The production source code is kept private to protect the full implementation, business logic, security-sensitive architecture, and project-specific assets.

## Live Project

https://zm-collection.vercel.app/

## What This Project Demonstrates

- Product browsing
- Cart persistence
- Theme support
- Admin dashboard screens
- Responsive storefront

## Tech Stack

- Next.js
- React
- TypeScript
- Radix UI
- Tailwind CSS
- Context API

## Public Repository Scope

This repository intentionally contains only a high-level folder outline and safe placeholder files. It is meant for portfolio review, project discovery, and architecture overview only.

The private source repository contains the complete application code, integrations, environment handling, database logic, authentication flows, and deployment configuration.
